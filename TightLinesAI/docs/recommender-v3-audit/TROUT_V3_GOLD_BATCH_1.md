# Trout V3 Gold Batch 1

This is the first archived 10-scenario trout truth set.

Purpose:
- establish a real trout benchmark
- stay river-only from the start
- pressure-test classic coldwater rivers and warmer tailwater contrast
- verify that streamer and minnow logic feels species-correct

## Batch Shape

- Scenario count: `10`
- River: `10`
- Clear: `7`
- Stained: `2`
- Dirty: `0`

This batch intentionally leans clear because trout most often punish sloppy presentation and bad color reads in clearer water.
Stained scenarios are included to make sure the engine can still become more visible without turning into bass logic.

## Core Regions

- Appalachian / tailwater trout
- Northeast freestone trout
- Mountain West trout
- Pacific Northwest trout
- Northern California trout

## Review Goals

For each scenario, verify:
- the top 1 is a believable trout start
- the top 3 stay inside a sensible streamer / minnow game plan
- colors feel realistic for the actual lure or fly type
- summer warm-tailwater trout tighten appropriately
- fall baitfish streamer windows open strongly when they should
