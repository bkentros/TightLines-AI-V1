# Smallmouth V3 Gold Batch 1

This is the first archived 10-scenario smallmouth truth set.

Purpose:
- establish a first real SMB benchmark
- cover both lake and river strongly
- cover winter, prespawn, spawn/postspawn, summer, fall
- pressure-test color lanes and water-type separation

## Batch Shape

- Scenario count: `10`
- Lake / pond: `5`
- River: `5`
- Clear: `6`
- Stained: `4`
- Dirty: `0`

This batch intentionally leans clear and stained because that is where smallmouth most often demand sharper tactical and color discipline.
Dirty-water SMB pressure can be added in later overlay batches.

## Core Regions

- Great Lakes / upper midwest
- Appalachian / warm highland rivers
- Northeast
- Midwest interior
- Mountain West

## Review Goals

For each scenario, verify:
- the top 1 is something a strong SMB angler would actually start with
- the top 3 stay within a believable smallmouth game plan
- colors feel realistic for the lure or fly type
- river scenarios stay current- and seam-aware
- lake scenarios stay rock / baitfish / craw / clean-water sensible
