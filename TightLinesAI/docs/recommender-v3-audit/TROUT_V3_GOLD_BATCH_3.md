# Trout V3 Gold Batch 3

This is the third archived 10-scenario trout truth set.

Purpose:
- stress the edges of trout river logic
- make sure streamer-first recommendations still hold in tougher shoulder-season and low-energy windows
- verify that colors remain disciplined and trout-specific across clear and stained water

## Batch Shape

- Scenario count: `10`
- River: `10`
- Clear: `8`
- Stained: `2`
- Dirty: `0`

This batch leans very clear because trout are most likely to expose weak logic there, especially in post-front, deep-winter, and late-fall tightening windows.

## Hard Windows Covered

- stained spring runoff-edge streamer control
- clear spring post-front restraint
- early-summer tailwater baitfish openings
- midsummer clear western crossover windows
- low-light late-summer trout minnow reads
- early-fall brown-window baitfish streamer pushes
- stained fall pulse visibility
- clear late-fall post-front tightening
- clear early-winter heavy-streamer control
- deep-winter subtle low-energy control

## Review Goals

For each scenario, verify:
- the top 1 still feels like a strong trout start, not just a technically valid one
- the top 3 remain trout-specific and streamer-minded
- colors stay realistic for sculpins, leeches, buggers, minnows, and articulated baitfish flies
- post-front and deep-winter trout stay bounded and subtle
- late-summer and early-fall windows open correctly without drifting into bass-style aggression
